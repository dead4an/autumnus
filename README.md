# 🍁 Autumnus

Automatic Differentiation and Neural Networks in Python.

## ❓ Why I made this repo?

The reason why I made this repo is learning. To better understand such frameworks as [PyTorch](https://github.com/pytorch/pytorch) or [Tensorflow](https://github.com/tensorflow/tensorflow) and deepen theoretical knowledges about neural networks I decided to implement autograd and various layers. I made it public so anyone intrested in these topics could always look at my implementations. For the same reason I'll try to write the code as clean as possible, since the above mentioned PyTorch and Tensorflow lack clarity.

This package **is not** intented to be used in commercial develompent or any kind of researches on NNs since some optimization techniques may not be implemented, the same goes for GPU acceleration.
