# Autumnus

Autumnus is a Python package implementing such things as

- Autograd engine
- Tensors (based on [NumPy](https://github.com/numpy/numpy))
- Neural network modules

and was inspired by [PyTorch](https://github.com/pytorch/pytorch) library.


# What is the goal?

This package was created for study purposes only and is not intended to be used in real-world cases. Here you won't find GPU acceleration and optimization techniques that are used in other libraries. The goal of this repo is to show how such libraries work.
